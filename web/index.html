Local tool. Talks straight to the mouse over WebHID using the reverse-engineered protocol. No website, no server.
Not connected. Use the 2.4G receiver (not Bluetooth), then click Connect.
Note: this mouse accepts key-macro writes but emits nothing. Use the raw-button options instead.
What this does: sets the side buttons to emit clean mouse buttons 4 & 5 (instead of the firmware's ⌘[/⌘] keystrokes), so any OS or remapper can bind them freely. Browsers/Finder treat 4 & 5 as Back/Forward automatically.
